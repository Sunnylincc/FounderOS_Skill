# mvp to launch workflow

## Goal
Drive a startup from one stage transition to the next with explicit gates.

## When to run
Run at stage boundary reviews.

## Inputs
- Prior stage artifacts
- Current constraints
- Team capacity

## Steps
1. Review assumptions and evidence.
2. Generate missing artifacts from templates.
3. Run devil’s advocate challenge.
4. Decide go / iterate / no-go.

## Outputs
- Updated stage docs
- Decision log
- Next 2-week execution plan

## Agent prompt
"Use this workflow to generate stage outputs with metrics, risks, and concrete next actions."

## Quality checklist
- Decision criteria explicit
- Risks have owners
- Metrics instrumented

## Common failure modes
- Skipping validation due to urgency
- Scope creep disguised as strategy
- Shipping without rollback criteria
