# Example — Real Estate Investment Tool

## Concept
Deal analyzer for small investors evaluating rental properties.

## Idea stage output
- Problem: spreadsheet-heavy underwriting with inconsistent assumptions.
- ICP: individual investors with 1–20 properties.
- Test: 20 user calls; validate willingness to pay for faster decisions.

## MVP scope
- Property input form
- Cap rate / cash-on-cash calculator
- Scenario comparison

## Launch safeguards
- Assumption transparency labels
- Data freshness indicators
- Manual verification reminder before purchase decisions

## Scale moves
- Portfolio dashboard
- Lender integration
- Team collaboration
